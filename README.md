# 🥔 iPatata

![iPatata Banner](https://img.shields.io/badge/Status-Version%201.1.0-success) ![Android](https://img.shields.io/badge/Platform-Android-green)

**iPatata** es la combinación perfecta entre la nostalgia de los reproductores clásicos y las consolas portátiles retro. Diseñado para Android, iPatata no solo te permite escuchar tu biblioteca musical de forma estilizada con su característica *Click Wheel*, sino que además incluye minijuegos clásicos como el mítico *Snake* para cuando quieras matar el tiempo.

Desarrollado por **ContratopDev**.

## ✨ Características Principales
- 🎶 **Reproductor Musical Retro**: Navegación por rueda clásica con sensibilidad ajustada a los gestos y menú inteligente que se adapta a lo que suena.
- 🕹️ **Gameboy Integrada**: Juega al mítico Snake con controles en pantalla estilo Gameboy (Cruceta, botones A/B de alto contraste automático, Select, Start).
- 📱 **Diseño Adaptativo**: Escala dinámicamente y mantiene sus proporciones perfectas en móviles de distintos tamaños y relaciones de aspecto.
- 🌈 **Visualizador de Audio**: Un ecualizador integrado en el "altavoz" inferior que reacciona de forma dinámica mientras se reproduce tu música.
- 🎨 **Temas Personalizables**: Cambia los colores de la interfaz a tu gusto al instante (Tema Gris, Negro, Poke, Hopps, Tema Contratop y Music Dynamics).
- 🔋 **HUD Nostálgico**: Barra superior con la hora exacta y un icono de batería hecho a medida.
- 🎧 **Crossfade**: Transición fluida de la duración que prefieras entre tus canciones favoritas.
- 🌟 **Nuevo Icono**: Renovado para la versión 1.1.0.

## 🚀 Instalación
Puedes descargar la última versión compilada (APK) desde la pestaña de **[Releases](../../releases)**.

1. Descarga el archivo `iPatata-1.1.0.apk`.
2. Pásalo a tu dispositivo Android o ábrelo directamente desde el navegador de tu móvil.
3. Si tu dispositivo te pide permisos para instalar aplicaciones de "Orígenes Desconocidos", dale a Aceptar.
4. ¡Disfruta de la patata!

## 🛠️ Desarrollo
Si quieres compilar la app tú mismo para trastear o mejorar el código:
1. Asegúrate de tener Android Studio instalado (Koala o superior).
2. Clona el repositorio: 
   ```bash
   git clone https://github.com/tu-usuario/iPatata.git
   ```
3. Ábrelo en Android Studio.
4. Sincroniza Gradle.
5. Dale al botón de **Play** (`Run 'app'`) teniendo un emulador o tu móvil conectado.
